Мобильный прототип- механика нанесения мейкапа по референсу Paper Dool DIY.

Состоит из 4 скриптов:
1) MakeUpData содержит в себе конфиги макияжа для кастомизации девочки(Scriptable Object);
2)GirlController - логика кастомизации девочки;
3)DraggableItem - содержит в себе настройки перетаскиваемых предметов и показатели выбора цвета( используется enum для удобного распознавания типа) ;
4)HandController- скрипт управления рукой, содержит основнуб логику управления , анимации и поведение в зависимости от выбранного предмета;

Стек технологий:
DOTWEEN - для управления и настройки всех анимаций в проекте;

Также создан адаптивный UI интерфейс для различных разрешений мобильных устройств;
